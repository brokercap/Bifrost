Redis 插件 参数解析

  记录被删除/更新时Redis的数据也会被同步更新/删除, 忽略query查询

Type

String : SET 命令写入数据

List : LPUSH 命令写入数据

Hash : HSET 命令写入数据

Zset : ZSET 命令写入数据

SET : SADD 命令写入数据

Key/FieldKey/Sort

Redis 写入数据时候的值,{$SchemaName},{$TableName},{$EventType},{$FiledName} 标签将会替换成数据库名 ，表名 ,事件类型 ， 甚至字段内容

Expir

仅支持类型为string 消息过期时间，必须为 int 类型，默认为0，不过期
