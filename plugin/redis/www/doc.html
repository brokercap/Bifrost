Redis 插件 参数解析

 

Type

key=>value : SET 命令写入数据

List : LPUSH 命令写入数据

DataType

string : 指定写入哪些数据, 选择这个，Value 参数生效

json : 自动将选择中的Fields 进行json 写入Redis

Key

Redis 写入数据时候的key值,{$SchemaName},{$TableName},{$EventType},{$"+filedName+"} 标签将会替换成数据库名 ，表名 ,事件类型 ， 甚至字段内容

Value

当DataType == string 的时候 , 同样会标签替换

Expir

消息过期时间，必须为 int 类型，默认为0，不过期
